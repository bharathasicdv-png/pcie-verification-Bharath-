# PCIe Design Verification Project

## Overview

This repository contains a complete PCI Express (PCIe) Design Verification project developed using SystemVerilog and UVM.

The objective of this project is to build a reusable and scalable verification environment for a PCIe Controller IP by following industry-standard ASIC Design Verification methodologies.

---

## Objectives

- Learn PCIe Controller Architecture
- Build a reusable UVM Verification Environment
- Develop SystemVerilog Testbench Components
- Verify PCIe Protocol Functionality
- Implement Assertions (SVA)
- Collect Functional Coverage
- Execute Regression Testing
- Practice Debugging using Verdi/DVE

---

## Scope

This project focuses on the functional verification of a PCIe Controller IP.

The repository demonstrates:

- SystemVerilog
- Universal Verification Methodology (UVM)
- Assertions
- Functional Coverage
- Scoreboard-based Checking
- Random Verification
- Regression Methodology
